library(vela) # Vela actuarial R package library(insuranceRating) library(tidyverse) # ── 1. Load portfolio data ────────────────────────── df <- vela_load("motor_exposure", ay = 2019:2024) |> mutate( age_band = cut(driver_age, c(17,25,35,45,60,Inf), right=FALSE), cc_band = cut(engine_cc, c(0,1000,1600,2000,Inf)), ncb = factor(ncb_yrs, levels=0:5), log_si = log(sum_insured) ) # ── 2. Frequency model (Poisson GLM) ──────────────── m_freq <- glm( claims ~ age_band + cc_band + ncb + zone + vehicle_use + log_si, family = poisson("log"), offset = log(exposure), data = df ) # ── 3. Severity model (Gamma GLM) ─────────────────── m_sev <- glm( avg_loss ~ age_band + cc_band + zone + repair_type, family = Gamma("log"), data = filter(df, claims > 0) ) # ── 4. Pure premium = Freq × Sev ──────────────────── df$pp <- predict(m_freq, df, type="response") * predict(m_sev, df, type="response") # ── 5. Validate: Gini, lift curve, one-way plots ──── vela_gini(df$pp, df$actual_loss_cost) # target: 0.40–0.55 vela_lift(m_freq, df, bands=10) vela_oneway(m_freq, "age_band", df) # marginal factor plot # ── 6. UBI telematics overlay (optional) ──────────── # Add telematics score as offset — no model re-fit needed df$pp_ubi <- df$pp * exp(df$tele_score_offset)

Key Precedents — Non-Disclosure / Material Facts
Pan Atlantic v Pine Top [1994] UKHL 8
Insurer won
House of Lords: insurer must prove non-disclosed fact induced them to write the risk — not merely that it was material. Establishes the inducement test still used today.

Versloot Dredging v HDI Gerling [2016] UKSC 45
Mixed
Supreme Court abolished fraudulent device rule. A fraudulent statement that doesn't affect claim outcome does NOT forfeit the entire claim — major shift for fraud defence strategy.

LIC of India v Asha Goel (2001) — Supreme Court of India
Policyholder won
Repudiation under Section 45 requires nexus between suppressed fact and cause of claim. Mere suppression without causal link insufficient to repudiate a life claim after 2 years.
